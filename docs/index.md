# Vagrant入門

## About

- Vagrantを使用するためのプラクティスに関するドキュメントです。
    - 最低限の使用方法が記載されており、すぐにVagrantを使用できるようになります。
- このドキュメントを読むと、以下の事が出来るようになります。
    - VagrantでVM作成を自動化することが可能です。
    - Vagrantの基本コマンドが理解できます。
    - `Vagrantfile`の仕組みについて理解できます。

### 想定読者

1時間ほどで完読できる内容となっております。

- Vagrantを初めて使用する方
- 手動でVMを作るのが面倒な方

### 動作環境

- Windows10
- Virtualbox
    - 6.1.14
- Vagrant
    - 2.2.10
